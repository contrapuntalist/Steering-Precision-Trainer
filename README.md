# Steering Precision Trainer

**Open the trainer:** https://contrapuntalist.github.io/Steering-Precision-Trainer/

A free, browser-based steering precision trainer for sim racers and real drivers.  
Works with **Simucube and most Direct Drive wheels** via the browser **Gamepad API**.

**Creator:** Instagram → https://instagram.com/bryceliracerpanda

---

## What it trains
- Steering accuracy/ fine muscle memory
- Holding the correct steering angle under time pressure
- Steering weak spots (after 120 degrees)

---

## How to use (30 seconds)
1. Plug in your wheel.
2. Open the trainer link above.
3. Press **SPACE** to spawn a target.
4. Match the target and hold within tolerance degree until it counts as a hit.

You’ll see **Last** and **Avg** reaction time at the top.

---

## Features
- Circular Rim mode (circular target) + Slider mode (linear Target)
- Adjustable tolerance + hold time (global)
- Random spawn timing (min/max delay)
- Lower-half targeting weak spots(left / right)
- Moving target mode
- Watermark + link to Instagram
